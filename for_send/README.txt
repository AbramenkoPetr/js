Войти директорию node_serv_auth.
Выполнить команду npm i.
Выполнить команду npm run start.
Запустится сервер авторизации на localhost:3001.
Войти директорию test_nextjs.
Выполнить команду npm i.
Выполнить команду npm run start или npm run dev.
Запустится основной сервер на localhost:3000.
В командной строке браузера ввести localhost:3000.
При этом отбразится домашняя страница из задания.
Для регисрации в хедере нажать Registration.
Если пользователь регистрируеся с email, который уже существует,
существующий удаляется и заменяется на новый.
Lля авторизации в хедере нажать Authorization.
Токен дополнительно сохраняется в localStorage под ключем tokenUser.
При этом пользователь остается авторизованным пока не будет авторизован
другой пользователь или ключ tokenUser не будет удален из localStorage.
